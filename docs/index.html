ReCalendar Timeblock Edition

A fork of ReCalendar with timeblock planner columns and a shutdown checkbox.

This is a personal fork of the excellent ReCalendar.js project by Igor Klimer. All the great features of the original are included — this fork simply adds a couple of extras for timeblock planning.

Create your calendar

What's different in this fork?

  • Timeblock planner columns — adds vertical lines to day pages creating a time column, two scheduling columns, and a notes column. Column widths are fully configurable.
  • Shutdown checkbox — adds a "Shut Down" text and checkbox to the header of day pages. Configurable per day of the week, so you can have it on workdays only.
